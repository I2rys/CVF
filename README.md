## Installation
Github:

    git clone https://github.com/I2rys/CVF
    
NPM Packages:
```
npm i axios && npm i chalk
```

## Usage

    node index.js <cms> <output>
    
+ cms - The cms on where to find vulnerabilities.
+ output - The output of the results.
